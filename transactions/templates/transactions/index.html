MPL LibStats

Reference Statistics Collection

Quick Start Guide

LibStats is a simple app that allows you to record, view, and generate reports about reference transactions that are completed in the library. You can access the app’s three functions by clicking the links in the menu bar above. A quick reference about reference transaction types is below if you need to refresh your memory from time to time. If you have any questions, please let your supervisor know.

Functions

  • Record: This is the main LibStats function. Use this function to record and add new reference transactions when you answer patron questions. Recorded transactions will be added to the transaction log for later viewing and reporting.
  • View: This function lets you review the recorded transaction log between a specified date range. This function also allows a portion of the full transaction log to be exported as a CSV.
  • Reports: This function lets you generate reference report based on a specified date range and a specific location (e.g. Circulation, Reference, or Children’s).

Reference Transaction Types

Directional

Simple questions about facilities, hours, etc.

For example: What time do you close? Where are the restrooms? Etc.

Information Services

General reference questions and questions about library cards or services.

For example: What are Kroger’s hours? How can I renew my card?

Digital Resources

Questions about accessing and using MPL digital resources.

For example: How do I use Libby and Hoopla? What is LawDepot? Can I use Ancestery at home?

Known-Item Requests

Patron request for specific items (known-items)

For example: Do you have The Martian by Andy Weir? Can I place a hold on this book?

Technology Help

Requests for help using public computers or other tech questions

For example: Can you help me print this document? Can you help me complete this application?
